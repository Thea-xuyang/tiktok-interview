# tiktok-interview
Data analysis projects for ByteDance International Payment Risk Control Intern Interview

## Project Overview
All projects are implemented by Python & SQL, focusing on mass data cleaning, abnormal sample detection, root cause analysis and KPI evaluation, matching the fraud monitoring work of payment risk control.

### 1. Multi-courier Route Optimization
Process hundreds of geographic delivery points, clean conflicting/repeated data with Python, use SQL multi-condition query to count indicators, output standardized analysis reports to optimize distribution efficiency.

### 2. Movie Recommendation System
Batch process user rating data, filter abnormal behavior via Pandas/Numpy, use SQL multi-table join to mine user features, encapsulate automated scripts to standardize data processing flow.

### 3. Machine Vision Target Recognition
Build simple neural network to identify objects from massive image datasets, filter abnormal blurred samples; collect multi-dimensional data to calculate recognition KPI and iterate model threshold.

## Technical Stack
- Database: SQL (multi-table join, group statistics, data extraction & abnormal positioning)
- Data Analysis: Python, Pandas, Numpy, basic machine learning neural network
- Office Tools: Excel PivotTable, PPT, Canva data visualization
